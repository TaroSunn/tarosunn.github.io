CSS布局模型

发表于 2017-11-17 | 分类于 笔记

流动（Flow)模型

网页默认布局模式，也就是说网页在默认状态下的HTML网页元素都是根据流动模型来分布页面内容的。
流动布局模型就有两个特征:

块级元素都会在所处的包含元素内自上而下按顺序垂直分布，在默认状态下，块状元素的宽为100%.

在流动模型中，内联元素都会在所处的包含元素内自左到右水平分布显示.

浮动（Float）模型

通过CSS的Float属性可以将块状元素并排显示

层布局模型（Layer）

绝对定位（position：absolute）

将元素从文档流中拖出来，然后使用left，right，top，bottom属性相对于其最接近的一个
具有定位属性的父包含块进行绝对定位。如果不存在这样的包含块，则相对于body元素，即相对于浏览器窗口。

相对定位（position： relative）

通过left,right,top,bottom属性确定元素在正常文档流中的偏移位置。相对定位完成的过程是首先按照static(float)方式生成一个元素(并且元素相层一样浮动了起来)，然后相对于以前的位置移动，移动的方向和幅度由left,right,top,bottom属性确定，偏移前的位置保持不动(偏移后位置没有被覆盖)。

固定定位（position:fixed）

固定定位的先对移动坐标是视图本身，不会随浏览器窗口的滚动条滚动而变化，所以固定定位会始终位于浏览器窗口内视图内的某个位置不会受文档流动影响。
