CSS三栏布局自适应

发表于 2017-11-21 | 分类于 笔记

前几天看了一个面试题：关于两边固定宽度中间自适应布局有几种实现方法。想到的就是用 Position Float Flex 这三个属性实现三种布局，从网上查了相关资料发现还有双飞翼布局和圣杯布局，所以总结一下。

基于Float属性布局

基于Position属性布局

大概思路是：利用 position:absolute 属性将左右两栏设置好，在用 margin 属性将中间一栏移动到中间

这种布局思路跟浮动布局差不多，利用position:absolute属性将左右两栏脱离文档流，从而实现固定的效果，而中间一栏也是利用 margin属性移动到中间。同样简单粗暴，与浮动布局有一样的问题，浏览器缩小到一定程度，会发生重叠。

基于Flex布局

Flex布局是css3里的一种新布局，又称“弹性布局”。需要将容器设置display:flex,两边设置好宽高，中间设置100%宽度，就能自动填充中间的区域。优点：当浏览器缩小到最小，布局不会发生变化。除了面向ie10一下不兼容以外，代码量和实现方式最好的一种布局。

圣杯布局和双飞翼布局

圣杯布局和双飞翼布局实现方式都是利用负margin和BFC来实现三栏布局，双飞翼布局的中间一栏嵌套了一个div
